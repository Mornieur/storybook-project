# TextAnimation

O `TextAnimation` é um componente de texto reutilizável usado para animar o título ou texto com transições suaves.

## Exemplo

```jsx
<TextAnimation type="title" isOpen={true} text="Text Example" />
```

### Parâmetros

Parâmetros disponíveis para personalizar o componente.

- type (obrigatório): Define o tipo de animação. Pode ser "title" para animar um título ou "text" para animar um parágrafo.
- isOpen (opcional): Define se a animação está aberta ou fechada. O valor padrão é false.
- text (opcional): O texto que será animado.
