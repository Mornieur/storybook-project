O `Button` é um componente de botão reutilizável usado para ações diversas.

### Exemplos:

<br />

#### Botão padrão

```jsx
<Container themeBtn={themeBtnTest.primary}>
  <SimpleContainer>
    <p>Default</p>
  </SimpleContainer>
</Container>
```

#### Botão Enviar

```jsx
<Container themeBtn={themeBtnTest.primary}>
  <SimpleContainer>
    <p>Enviar</p>
  </SimpleContainer>
</Container>
```

#### Botão "Apagar"

```jsx
<Container themeBtn={themeBtnTest.primary} onClick={(value: any) => {}}>
  <SimpleContainer>
    <p>Apagar</p>
  </SimpleContainer>
</Container>
```
